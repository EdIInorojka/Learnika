# Codex prompt — Wave 3

 

Implement diagnostics, mastery and the weekly learning loop on top of approved Wave 2 evidence.

 

Required outcomes:

 

- seed versioned mathematics skill graph;

- create an original diagnostic for selected skills;

- implement mastery state with uncertainty, recency and evidence weights;

- prevent single-answer mastery changes;

- implement school, restorative and target tracks;

- implement constrained weekly planner and spaced review;

- add first priority textbook mapping workflow;

- generate concise weekly parent report;

- include input-mode and independence summaries without transcript bodies;

- instrument activation, transfer, retention, voice value and report metrics;

- add experiment framework for plan and report variants.

 

Do not expand to new grades, subjects or native mobile.

 

Acceptance criteria:

 

- diagnostic and homework evidence use the same canonical skills;

- changing textbook preserves mastery;

- plan explains why each item is present;

- report answers what improved, what is difficult and what happens next;

- report does not expose unnecessary voice transcripts;

- mastery policy is covered by scenario tests;

- curriculum and independent review approve.
