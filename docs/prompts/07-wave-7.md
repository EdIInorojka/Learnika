# Codex prompt — Wave 7

 

Start only after the school-product business gate is approved and design-partner schools are named.

 

Implement a bounded school beta.

 

Required outcomes:

 

- organization, school, academic year, class, enrollment and teacher assignment;

- explicit school versus family authorization;

- roster import and invite codes;

- teacher assignment and assessment builder;

- online delivery with time and attempt settings;

- class skill analytics;

- PDF variants, answer keys and answer sheets;

- OMR prototype with confidence and manual-review queue;

- rubric-assisted open-response review with teacher confirmation;

- school license and entitlement model;

- audit, export and retention controls;

- pilot measurement of teacher time and renewal intent.

 

Teacher voice comments remain outside the first school beta unless a separate approved task activates them.

 

Do not build a full electronic diary, attendance, timetable, HR or government integration platform.

 

Release requires security isolation tests, teacher workflow validation, restore readiness and independent-review approval.
